# Trip Planner Web App 

# Technologies Used:-
    OpenAI :- https://openai.com

    React Js :- https://react.dev


# Steps to install

    1- Run npm install in Project root directory.
    2- Add openAI Authorization Bearer token to App.js.
    3- Run npm start.
